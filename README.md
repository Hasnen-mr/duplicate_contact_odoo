# duplicate_contact_odoo

Smart Duplicate Contact Manager for Odoo — detect, review, and merge duplicate contacts with fuzzy matching, confidence scores, and a merge wizard.

## Odoo version branches

| Branch | Odoo |
|--------|------|
| `14.0` | Odoo 14 |
| `15.0` | Odoo 15 |
| `16.0` | Odoo 16 |
| `17.0` | Odoo 17 |
| `18.0` | Odoo 18 |
| `19.0` | Odoo 19 |

## Install

1. Clone the branch matching your Odoo version.
2. Add the repo root (or `duplicate_contact` folder) to your `addons_path`.
3. Update Apps list and install **Smart Duplicate Contact Manager**.
